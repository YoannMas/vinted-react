## Dependencies

- axios
- fontawesome
- material UI
- stripe
- js-cookie
- node-sass
- react-multi-carousel


## Setup instructions

Clone this repository 

```
git clone https://github.com/YoannMas/vinted-react.git
```

Install dependencies

```
yarn install
```

Run it

```
yarn start
```
